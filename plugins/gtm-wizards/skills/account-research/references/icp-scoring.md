# ICP Scoring Reference

## How to Use This File
This file provides the LOCAL scaffold for ICP scoring. ALWAYS search Notion for "ICP" and "Ideal Customer Profile" to get current scoring criteria before scoring any account.

## ICP Categories (High-Level)

### Tier 1 — Ideal Fit
- **Industry**: Manufacturing, Distribution, Retail, Healthcare, Financial Services
- **Revenue**: $500M–$5B
- **ERP**: SAP, Oracle, NetSuite, Microsoft Dynamics
- **AP Volume**: 50,000+ invoices/year
- **Pain Signals**: Manual AP processing, high cost-per-invoice, DPO lagging peers, compliance pressure

### Tier 2 — Good Fit
- **Revenue**: $100M–$500M
- **AP Volume**: 10,000–50,000 invoices/year
- **Has budget cycle**: Active initiative or RFP in progress

### Tier 3 — Exploratory
- **Revenue**: $50M–$100M
- **Some AP pain but not yet a priority**
- **Longer sales cycle expected**

## Scoring Criteria
- Invoice volume: High/Medium/Low
- ERP environment: Compatible/Unknown/Incompatible
- Financial pain signals: Strong/Moderate/Weak
- Competitive landscape: No incumbent/Incumbent vulnerable/Incumbent entrenched
- Budget timing: Active/Planned/Unknown
